# NeonCounter
Маленькая desktop-программа-счётчик на Python (Tkinter):
- окно всегда поверх всех окон;
- верхняя строка — желаемое значение (цель);
- нижний ввод — дневное значение (`+` или `-`), которое добавляется к текущему;
- визуальный прогресс в виде шкалы;
- выбор неонового цвета.
